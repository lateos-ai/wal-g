# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please email security@lateos.ai instead of using the issue tracker.

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will respond within 24 hours for critical issues.

## Security Commitments

- **Critical CVEs (CVSS 9.0+):** Fixed within 24 hours
- **High CVEs (7.0–8.9):** Fixed within 7 days
- **Medium CVEs (4.0–6.9):** Fixed within 30 days

## Supported Versions

- Latest version: Fully supported
- Previous version: Security updates only
- Older versions: No support (upgrade recommended)
